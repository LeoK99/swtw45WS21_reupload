Über die Sheet Datei können Funktionen (z.B. createUser(String userName, String password)) angefordert werden, die für Aufrufe aus dem Frontend benötigt werden.
Wenn die Funktion nicht selbst erklärend ist, kann optional eine Beschreibung eingefügt werden.

In der Feature-List.adoc Datei sind alle Frontend-Features die noch implementiert werden müssen enthalten.
